# 💰 Calculadora de Investimentos

Aplicação web para simulação de investimentos com foco em **visualização clara da evolução do capital** ao longo do tempo.

Permite analisar o impacto de **aportes periódicos**, **taxa de retorno** e **tempo** sobre o crescimento do patrimônio.

---

## 🌐 Acessar o projeto

[🔗 Clique aqui para acessar](https://aleraczinoski.github.io/CalculadoraInvestimentos/)

---

## ⚙️ Stack

* **Node.js** — gerenciamento de pacotes (npm)
* **Vite** — build tool e ambiente de desenvolvimento
* **JavaScript (ES6+)** — lógica da aplicação
* **HTML5** — estrutura da aplicação
* **TailwindCSS** — estilização
* **Chart.js** — visualização de dados

---

## 📊 Funcionalidades

* Simulação de investimentos com **juros compostos**
* Configuração de:

  * valor inicial
  * aportes mensais
  * taxa de rendimento
  * período (meses/anos)
* Visualização de:

  * evolução do patrimônio
  * comparação entre **capital investido vs rendimento**
* Gráficos dinâmicos e interativos

---

## 📦 Rodando localmente

```bash
git clone https://github.com/aleraczinoski/CalculadoraInvestimentos.git

cd CalculadoraInvestimentos

npm install

npm run dev
```

---

## 🏗️ Build

```bash
npm run build
```

Gera a pasta `dist/` com os arquivos otimizados para produção.
